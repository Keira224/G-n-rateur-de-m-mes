# Studio Meme Maker

Mini-projet d'admission construit avec React, Vite, JavaScript, CSS moderne, `localStorage` et `html2canvas`.

## Lancer le projet

```bash
npm install
npm run dev
```

## Construire la version de production

```bash
npm run build
```

## Fonctionnalites principales

- upload d'image locale
- texte en haut et en bas du mème
- aperçu en temps reel
- telechargement PNG
- sauvegarde dans `localStorage`
- galerie locale avec suppression
- partage via l'API Web Share si le navigateur le permet
